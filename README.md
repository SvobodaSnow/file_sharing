# File Sharing
 
## Серверная часть приложения обмена данными для организации работы компании

## Описание

Приложение разработано для целей обучения программированию серверной части на языке Python.
Основной идеей для создания приложения послужила проблема обмена данными компании, работа которой осуществляется по всей стране. Так как работа велась с разных мест то необходима была система организации данных.
##
## Функционал программы:

	- Работа с пользователями:
		- Создание пользователя;
		- Получение пользователя по идентификатору;
		- Поиск и получение пользователей по имени;
		- Поиск и получение пользователей по адресу электронной почты;
		- Поиск и получение пользователей по должности;
		- Поиск и получение пользователей по подразделению;
		- Поиск и получение пользователей по дате трудоустройства;
		- Изменение данных пользователя;
		- Удаление пользователя.
	- Работа с подразделениями:
		- Добавление нового подразделения;
		- Редактирование названия подразделения;
		- Удаление подразделения.
	- Работа с должностями:
		- Добавление новой должности;
		- Редактирование названия должности;
		- Удаление должности.
	- Работа с задачами:
		- Создание задачи;
		- Добавление нового файла к задаче;
		- Получение задачи по идентификатору;
		- Получение всех задач;
		- Изменение данных задачи;
		- Закрытие задачи;
		- Удаление задачи.
	- Работа с проектами:
		- Добавление нового проекта;
		- Редактирование названия проекта;
		- Удаление проекта.
	- Работа со статусами проектов:
		- Добавление нового статуса;
		- Редактирование названия статуса;
		- Удаление статуса.
